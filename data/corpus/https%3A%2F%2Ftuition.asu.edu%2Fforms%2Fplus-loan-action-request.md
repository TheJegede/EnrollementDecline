# Parent PLUS Loan Action Request

## Purpose

This eForm is used to reduce or cancel an accepted and/or a disbursed Parent PLUS loan. You (the student) will need to submit a request that generates a MyASU Task for your parent to submit the Parent PLUS Loan Action Form.

This eForm should only be submitted by the student. If you are a parent please have your student submit this eForm and monitor your MyASU for the Parent PLUS Loan Action Form Finance Task, which can be submitted to reduce or cancel the Parent PLUS Loan you accepted.

## Instructions

Submit the eForm.
