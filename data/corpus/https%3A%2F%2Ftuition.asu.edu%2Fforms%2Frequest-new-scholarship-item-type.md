# Request for new scholarship item type

For use by ASU personnel only. This form is used by ASU departments to request a Scholarship Item Type (SIT) for a newly established scholarship. A SIT is a 12-digit identifier used as a funds pass-through account for disbursement to students. Funds are transferred to the SIT in a number of ways, usually from an ASU Foundation account. For more details, see the Scholarship Award Funding policies and procedures.

## Instructions

- Complete the online form which will be submitted to Financial Aid and Scholarship Services. Please allow 3-5 business days for requests to be processed.

**Scholarship Award Funding Policy & Procedures**

Please read the updated funding policy and procedures for departments awarding scholarships through the new electronic system.
